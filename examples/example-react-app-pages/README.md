# React + TypeScript + Vite

## Steps to setup the example-react-app-pages

### Configure

- update the .env file for the below config values

```env
VITE_EXP_SPACE_EDITOR_URL = https://domain.com/ui/editor

VITE_TRIDION_SITES_GRAPHQL_URL = https://domain.com/cd/api

VITE_TRIDION_SITE_URL = https://SiteUrl.com
```

### Installation
- npm install


### Run the app
- npm run dev
